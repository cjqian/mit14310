The Binomial Distribution

The binomial distribution is the number of successes in a sequence of nnn independent trials (sum of Bernoullis).

If X1…XnX_1 \dots X_nX1​…Xn​ are I.I.D. Bernoulli with success ppp, X=∑k=1nXk∼B(n,p)X = \sum_{k=1}^n{X_k \sim B(n, p)}X=k=1∑n​Xk​∼B(n,p)

The PDF: f(x;n,p)=Pr(X=x)=(nx)px(1−p)n−x for x=0,1,2,3…nf(x; n, p) = Pr(X=x) = {n \choose x}p^x(1-p)^{n-x} \text{ for } x = 0, 1, 2, 3 \dots nf(x;n,p)=Pr(X=x)=(xn​)px(1−p)n−x for x=0,1,2,3…n

Shape is symmetric and bell-shaped.

The Hypergeometric Distribution

The binomial distribution is the # of successes in a sample of n with replacement. The hypergeometric distribution has no replacement. (For example, the number of unique pizza toppings, number of balls taken from an urn).

Let N=A+BN = A + BN=A+B where AAA is successes and BBB is failures.

f(X∣A,B,n)=(Ax)(Bn−x)(A+Bn)f(X | A, B, n) = \frac{{A \choose x}{B \choose {n-x}}}{{A+B \choose n}}f(X∣A,B,n)=(nA+B​)(xA​)(n−xB​)​

Then, E(X)=nAA+BE(X) = \frac{nA}{A+B}E(X)=A+BnA​ and V(X)=n(AA+B)(BA+B)(A+B−nA+B−1)V(X) = n(\frac{A}{A+B})(\frac{B}{A+B})(\frac{A+B-n}{A+B-1})V(X)=n(A+BA​)(A+BB​)(A+B−1A+B−n​)

Notice that this is similar to the binomial, with p=AA+Bp = \frac{A}{A+B}p=A+BA​ and q=BA+Bq = \frac{B}{A+B}q=A+BB​.

The Poisson Distribution

This discrete distribution expresses the probability of # events in a fixed interval of time if

  1. The event can be counted in whole numbers
  2. The occurences are independent
  3. The average frequency of occurence for a time period is known

Examples: # of buses in a bus station during a fixed time, cases of flu during a day, goals scored during a match.

  • Not the number of people showing up to a cafeteria during a given time, because people tend to arrive together.

Formally, for t \geq 0, let $N_t% be integer-valued random variables.

  3. NsN_sNs​ and Nt+s−NtN_{t+s} - N_tNt+s​−Nt​ have identical distribution (not time-dependent)

  5. limt→0P(N>1)t=0lim_{t\rightarrow 0}\frac{P(N>1)}{t}=0limt→0​tP(N>1)​=0 (no simultaneous arrival)

Then for any non-negative integer kkk,

P(Nt=k)=(γt)ke−γtk!P(N_t = k) = \frac{(\gamma t)^ke^{-\gamma t}}{k!}P(Nt​=k)=k!(γt)ke−γt​

Representing Poisson Distributions

If you divide the interval [0, ttt] into nnn subintervals (s.t. the probability of two occurences in each subinterval is 0), the probability of success in each subinterval is then λn\frac{\lambda}{n}nλ​, which is approximately binomial!

Exponential Distributions

Exponential is the waiting time between two events in a Poisson process.

fx=λe−λxf_x = \lambda e^{-\lambda x}fx​=λe−λx if x > 0 and 0 otherwise.

Then, E(X)=1lambdaE(X) = \frac{1}{lambda}E(X)=lambda1​, and V(X)=1lambda2V(X) = \frac{1}{lambda^2}V(X)=lambda21​. This is memoryless (doesn’t matter what has happened before, only depends on the current location of the interval)!

  • Continuous
  • Shape: bell-shaped, symmetrical.
  • Integral of the binomial
  • E(Z)=0;V(Z)=1E(Z) = 0; V(Z) = 1E(Z)=0;V(Z)=1

The normal distribution: finding the area under the curve

  • No closed-form integral; we use a Z-score table instead to find the probability of a value below a given value

Some R functions

  • rnorm(100, 3, .25) generates 1000 numbers with mean 333 and std. .25.25.25
  • dnorm(0, 0, 5) gives PDF height of normal with mean=0 and std. = .5
  • pnorm(1.96, 0, 1) gives area under the standard normal curve to the left of 1.96 (~.0975)
    • To the right, pnorm(1.96, lower.tail=False)
  • qnorm goes opposite of pnorm; qnorm(0.975, 0, 1) = ~1.96
