Set theory and probability

Definitions

  • Sample space (S) is a collection of all possible outcomes of an experiment
  • Event (A, B, C) is any collection of outcomes
    • Individual outcomes
    • Entire sample space
    • Null set
  • If an outcome is in an event, it has occurred
  • If event A contains event B, all outcomes in B are also in A

Other set theory results

  • If A∈CA \in CA∈C, A∪B=BA \cup B = BA∪B=B
  • If A∈BA \in BA∈B and BinAB in ABinA, A=BA = BA=B
  • If A∈BA \in BA∈B, AB=AAB = AAB=A
  • A∪Ac=SA \cup A_c = SA∪Ac​=S

Differences between probability and set theory

  • Notation
    • In probability theory, we use ABABAB instead of A∩BA \cap BA∩B
  • Terminology
    • A and B are mutually exclusive if they have no outcomes in common (instead of disjoint)
    • A and B are exhaustive if A∪B=SA \cup B = SA∪B=S (instead of complementary)

Simple sample space example

Suppose we have a finite sample space (finite outcomes) and a function n that gives the number of elements in a set.

Define function P(A)=n(A)n(S)P(A) = \frac{n(A)}{n(S)}P(A)=n(S)n(A)​. How can we check that this is a probability?

  1. Because it’s a count, it’s always non-negative.
  2. P(S) = 1 by definition.
  3. P(A∪B)=n(A∪B)n(S)=n(A)n(S)+n(B)n(S)=P(A)+P(B)P(A \cup B) = \frac{n(A \cup B)}{n(S)} = \frac{n(A)}{n(S)} + \frac{n(B)}{n(S)} = P(A) + P(B)P(A∪B)=n(S)n(A∪B)​=n(S)n(A)​+n(S)n(B)​=P(A)+P(B)

We can use this simple sample space framework for different probabilistic experiments. For example:

Say you roll 2 fair dice and would like to compute the probability that the sum on the faces = 4. We can count the possible outcomes (n(A) = 3 (1 and 3, 2 and 2, 3 and 1)); there are 36 total outcomes. So, P(A)=336=112P(A) = \frac{3}{36} = \frac{1}{12}P(A)=363​=121​.

Sampling examples

Say Massachusetts offers license plates with 6 different characters (alphanumeric).

Sampling with replacement

What’s the probability of an all-digit license plate?

  • n(S) = 36 possibilities for each of 6 charaacters = 36636^6366
  • n(A) = 10610^6106
  • P(A) = 106366=.0005\frac{10^6}{36^6} = .0005366106​=.0005

Sampling without replacement

What if they don’t reuse digits on the plate?

  • n(S) = 36!30!\frac{36!}{30!}30!36!​
  • n(A) = 10!4!\frac{10!}{4!}4!10!​
  • P(A) = .0001

Ordered and unordered arrangements

  1. If experiment has two parts, one with mmm outcomes and the other with nnn outcomes, the experiment has in total mxnmxnmxn outcomes.
  2. Any ordered arrangement of objects is a permutation (sampling without replacement).
    • # of permutations of N = N!
    • # of permutations of n objects from N is N!(N−n)!\frac{N!}{(N-n)!}(N−n)!N!​
  3. Any unordered arrangement of objects is a combination
    • # of combinations of n objects from N is N!(N−n)!n!\frac{N!}{(N-n)!n!}(N−n)!n!N!​ = (Nn){N \choose n}(nN​)

Independence and a basketball example

  • Events A and B are independent if P(AB)=P(A)P(B)P(AB) = P(A)P(B)P(AB)=P(A)P(B).
    • Be wary of intuition. Example: suppose you toss a dice. AAA is the event that you roll < 5. BBB is the event that you roll an even number. These events are independent by this definition.
    • Intuition: knowing that one event occured doesn’t tell you anything about the probability that another event occured.
  • If A and B are independent, A and BcB_cBc​ are also independent.

Conditional probability

In the case where A and B are not independent, the probability of A conditional on B, P(A∣B)P(A|B)P(A∣B), is P(AB)P(B)\frac{P(AB)}{P(B)}P(B)P(AB)​, given P(B)>0P(B) > 0P(B)>0.

  • Numerator: redefining only the part of the event that is relevant given that B has occurred
  • Denominator: redefining the sample space now that B has occurred

Visual: given a Venn diagram with A and B in S, numerator is the intersection and denominator is the space of B.

P(A)=P(A∣B)P(B)+P(A∣Bc)P(Bc)P(A) = P(A|B)P(B) + P(A|B_c)P(B_c)P(A)=P(A∣B)P(B)+P(A∣Bc​)P(Bc​)

Bayes’s Law: If A and B are independent, P(A∣B)=P(A)P(A | B) = P(A)P(A∣B)=P(A).

Zika example

1 in 1000 people get Zika virus; the test gives positive reading .99 if they have it, and .05 if the person does not. What’s the probability that they actually have the virus given the test?

Formalizing:

  • P(Z)=.001P(Z) = .001P(Z)=.001
  • P(Zc)=.999P(Z_c) = .999P(Zc​)=.999
  • P(+∣Z)=.99P(+|Z) = .99P(+∣Z)=.99
  • P(+∣Zc)=.05P(+|Z_c) = .05P(+∣Zc​)=.05
    P(Z∣+)=P(+∣Z)P(Z)P(+∣Z)P(Z)+P(+∣Zc)P(Zc)=.019P(Z|+) = \frac{P(+|Z)P(Z)}{P(+|Z)P(Z) + P(+|Z_c)P(Z_c)} = .019 P(Z∣+)=P(+∣Z)P(Z)+P(+∣Zc​)P(Zc​)P(+∣Z)P(Z)​=.019

Intuition: if the test was perfect, the probability of testing positive is .001. The test is valuable but not perfect, so the probability doesn’t get updated too much as a result.
