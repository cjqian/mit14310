Introduction to random variables

A random variable is a real-valued function whose domain is the sample space.

  • A probability goes from sample space to the unit interval (values from 0 to 1)
  • A random variable goes from the sample space to the real line
  • A probability induces the distribution of the random variable

Different types of random variables:

  • Discrete: finite/countably infinite values
  • Continuous: any value in a bounded/unbounded interval of the real line

Probability functions of random variables
